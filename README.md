# Odin-restaurant-page
This is a restaurant page that I created using HTML, CSS and JavaScript with the help of webpack for bundling purposes.<br/>
The project was inspired by <NAME>'s [Odin Project](https://www.theodinproject.com/) curriculum on Web Development

## Installtion 
To run this application you need to clone or download it from GitHub then open your terminal in the root directory where all files are located (index.html
and style.css) and type `npm i` followed by `npx webpack`. Then go back into index.html file and click anywhere inside it and open it with browser
